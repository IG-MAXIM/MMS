Определение функций в Python

Научитесь создавать собственные функции с помощью def, передавать параметры и возвращать результат

Что такое функция?

Функция — это именованный блок кода, который можно вызывать много раз. Она помогает избежать повторений и делает программу структурированной.

Функция = Рецепт Один раз записал, используешь многократно
Вызов = Приготовление Передал продукты → получил блюдо

Функции позволяют:

  • Уменьшить дублирование кода
  • Разбить задачу на логические части
  • Повторно использовать готовые решения

Синтаксис def

Функция объявляется с помощью ключевого слова def, за ним следует имя, круглые скобки с параметрами и двоеточие. Тело функции пишется с отступом.

def имя_функции(параметр1, параметр2):
    """Документация (необязательно)"""
    результат = ...
    return результат

Функция может ничего не возвращать (тогда return не обязателен).

def greet(name):
    print("Привет,", name)

greet("Анна") # Вызов функции

Параметры и аргументы

Параметры — переменные, указанные в скобках при объявлении функции. Аргументы — конкретные значения, передаваемые при вызове.

Позиционные

def power(base, exp):
    return base ** exp

# Аргументы по порядку
power(2, 3) # 8

Именованные

# Можно явно указать имена
power(exp=3, base=2) # 8

Также можно задавать значения по умолчанию:

def greet(name="Гость"):
    print("Привет,", name)

greet() # Привет, Гость
greet("Анна") # Привет, Анна

Практические задания

Задание 1: Функция приветствия

Напишите функцию hello(), которая принимает имя и выводит "Привет, [имя]!". Вызовите её для трёх разных имён.

def hello(name):
    print(f"Привет, {name}!")

hello("Анна")
hello("Борис")
hello("Вера")

Задание 2: Функция с возвратом

Создайте функцию max_of_two(a, b), которая возвращает большее из двух чисел. Протестируйте её.

def max_of_two(a, b):
    if a > b:
        return a
    else:
        return b

print(max_of_two(7, 12)) # 12

Задание 3: Значения по умолчанию

Напишите функцию multiply(a, b=1), которая возвращает произведение двух чисел. Если второй аргумент не указан, он должен быть равен 1. Проверьте оба варианта вызова.

def multiply(a, b=1):
    return a * b

print(multiply(5, 3)) # 15
print(multiply(5)) # 5

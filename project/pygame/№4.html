Переменные и типы данных в Python

Освойте работу с переменными, типами данных и базовыми операциями

Переменные

Что такое переменная?

Переменная – это именованная область памяти, в которой можно хранить данные. Представьте себе коробку с биркой: на бирке написано имя, а внутри лежит значение.

Имя переменной

Уникальное название, по которому мы обращаемся к данным. Например: age, name, price.

Значение

Данные, которые хранятся в переменной. Может быть числом, строкой, списком и т.д. Например: 15, "Анна".

Правила именования:

  • Только буквы, цифры и знак _
  • Не может начинаться с цифры
  • Регистр важен: name и Name – разные переменные

Основные типы данных

Python автоматически определяет тип данных при присваивании. Вот главные из них:

int
целые числа
15, -3, 0
float
дробные числа
1.65, -0.5
str
строки
"Привет"
bool
логические
True / False

Проверить тип можно функцией type():

print(type(15))
# <class 'int'>

Важные операции: // и %

Целочисленное деление //

Возвращает целую часть от деления.

15 // 2 = 7
123 // 10 = 12

Полезно: отбросить последние цифры числа.

Остаток от деления %

Возвращает остаток после целочисленного деления.

15 % 2 = 1
123 % 10 = 3

Полезно: получить последнюю цифру числа.

Другие полезные операции: / и **

Обычное деление /

Возвращает вещественный результат (float).

15 / 2 = 7.5
10 / 4 = 2.5

Важно: результат всегда float, даже если числа делятся нацело.

Возведение в степень **

Вычисляет степень числа.

2 ** 3 = 8
5 ** 2 = 25
9 ** 0.5 = 3.0

Полезно: можно извлекать корни, используя дробные степени.

Практические задания

Задание 1: Базовые переменные

Создайте переменные для имени, возраста и роста, затем выведите их на экран.

name = "Анна"
age = 15
height = 1.65
print("Имя:", name)
print("Возраст:", age)
print("Рост:", height)

Задание 2: Простые вычисления

Посчитайте сумму двух чисел и удвоенный возраст через переменные.

a = 10
b = 5
sum_ab = a + b
double_age = age * 2
print("Сумма:", sum_ab)
print("Удвоенный возраст:", double_age)

Задание 3: Строки

Соедините две строки и выведите приветствие.

greeting = "Привет"
name = "Анна"
message = greeting + ", " + name + "!"
print(message)

Задачи на разряды числа

Сумма цифр двузначного числа

Вводится двузначное число. Найти сумму его цифр.

n = int(input("Двузначное число: "))
tens = n // 10
units = n % 10
print("Сумма цифр:", tens + units)

Перестановка цифр

Поменять местами цифры двузначного числа (было 12 → 21).

n = int(input("Двузначное число: "))
tens = n // 10
units = n % 10
reversed = units * 10 + tens
print("Перевёрнутое:", reversed)

Цифры трёхзначного числа

Вывести отдельно сотни, десятки и единицы для трёхзначного числа.

n = int(input("Трёхзначное число: "))
hundreds = n // 100
tens = (n // 10) % 10
units = n % 10
print("Сотни:", hundreds)
print("Десятки:", tens)
print("Единицы:", units)

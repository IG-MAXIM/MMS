Возврат значения (return) и области видимости переменных

Углубитесь в работу функций: возврат результатов, локальные и глобальные переменные

Оператор return

return завершает выполнение функции и возвращает указанное значение в место вызова. Если return не указан, функция возвращает None.

def square(x):
    return x * x

result = square(5)
print(result) # 25

Можно возвращать несколько значений через запятую — они автоматически упаковываются в кортеж.

def min_max(a, b):
    return (a, b) if a < b else (b, a)

mn, mx = min_max(7, 3)
print(mn, mx) # 3 7

Локальные и глобальные переменные

Локальные переменные создаются внутри функции и видны только в ней. Глобальные переменные объявлены на уровне модуля и доступны везде.

Локальная

def func():
    x = 10
    print(x)
func() # 10
print(x) # Ошибка!

Глобальная

counter = 0
def increment():
    global counter
    counter += 1
increment()
print(counter) # 1

Для изменения глобальной переменной внутри функции нужно объявить global. Без этого создастся локальная переменная с тем же именем.

Практические задания

Задание 1: Возврат значения

Напишите функцию cube(n), которая возвращает куб числа. Протестируйте её для нескольких значений.

def cube(n):
    return n ** 3

print(cube(3)) # 27
print(cube(5)) # 125

Задание 2: Несколько возвращаемых значений

Создайте функцию sum_and_diff(a, b), которая возвращает сумму и разность чисел a и b. Присвойте результаты переменным и выведите их.

def sum_and_diff(a, b):
    return a + b, a - b

s, d = sum_and_diff(10, 4)
print("Сумма:", s, "Разность:", d)

Задание 3: Глобальный счётчик

Объявите глобальную переменную call_count = 0. Напишите функцию track_call(), которая увеличивает её на 1 при каждом вызове. Вызовите функцию 3 раза и выведите итоговое значение счётчика.

call_count = 0

def track_call():
    global call_count
    call_count += 1
    print(f"Вызов номер {call_count}")

track_call()
track_call()
track_call()
print("Всего вызовов:", call_count) # 3

Задание 4: Локальная vs глобальная

Вне функции задайте переменную value = 100. Внутри функции change_value() попробуйте изменить её без global. Выведите value до и после вызова функции и объясните результат.

value = 100

def change_value():
    value = 200 # Создаётся локальная переменная
    print("Внутри функции:", value)

print("До вызова:", value) # 100
change_value() # Внутри: 200
print("После вызова:", value) # 100 (глобальная не изменилась)

Главная идея: Понимание областей видимости и возврата значений — ключ к написанию надёжных и предсказуемых программ. Чёткое разделение ответственности между функциями делает код модульным и тестируемым.

Проект «Калькулятор с меню»

Создайте многофункциональный консольный калькулятор с текстовым меню, который использует функции, циклы и глобальное состояние

Постановка задачи

Разработайте программу-калькулятор, которая работает в бесконечном цикле и предлагает пользователю меню с выбором операции. После выбора операции программа запрашивает числа (одно или два, в зависимости от операции) и выводит результат. Результат сохраняется и может быть использован в следующем вычислении.

Что нужно реализовать

  • Вывод меню с перечнем доступных операций (сложение, вычитание, умножение, деление, возведение в степень, целочисленное деление, остаток от деления, выход)
  • Ввод номера операции от пользователя
  • Ввод чисел (одного или двух) в зависимости от выбранной операции
  • Вычисление результата с помощью отдельных функций для каждой операции
  • Сохранение последнего результата в глобальной переменной
  • Возможность использовать предыдущий результат как первый операнд (специальный пункт меню или автоматически при вводе пустой строки)
  • Обработку ошибок: деление на ноль, ввод некорректных символов, несуществующий пункт меню

Требования к реализации

Технические требования

  • Программа содержит не менее 4 отдельных функций для разных арифметических операций
  • Используется глобальная переменная для хранения последнего результата
  • Реализован бесконечный цикл while True с выходом через break
  • Пользовательский ввод проверяется на корректность (try/except или проверка типа)
  • Код разбит на логические блоки, функции имеют понятные имена и, при необходимости, строки документации

Критерии качества

  • Удобное и понятное меню
  • При делении на ноль выводится предупреждение, программа не завершается аварийно
  • При вводе букв вместо чисел программа сообщает об ошибке и даёт повторить ввод
  • Результат предыдущего вычисления корректно используется
  • Код легко читается и не содержит дублирования

Перед отправкой проекта убедитесь, что:

Функциональность

  • Все арифметические операции работают корректно

Код

  • Программа содержит отдельные функции для операций
  • Использована глобальная переменная для хранения последнего результата
  • Код хорошо структурирован, есть комментарии

  • Файл сохранён как menu_calc.py

Проект «Анализатор текста»

Итоговый проект по обработке строк и списков: создайте программу, которая исследует текст и выдаёт полезную статистику

Постановка задачи

Вы разрабатываете консольный анализатор текста — программу, которая принимает от пользователя произвольный текст и выводит его характеристики.

Обязательный минимум

Программа должна вывести не менее четырёх различных показателей текста. Например:

  • Количество символов (с пробелами / без пробелов)
  • Количество слов
  • Количество предложений
  • Самое частое слово (если таких несколько — любое)

Вы можете добавить и другие метрики: самое длинное/короткое слово, среднюю длину слова, количество знаков препинания и т.д.

Сценарий работы

  1. Пользователь вводит текст (можно многострочный — на ваше усмотрение).
  2. Программа обрабатывает текст и выводит статистику в читаемом виде.
  3. После вывода программа может спросить, не хочет ли пользователь проанализировать другой текст, или завершиться.

Требования к реализации

Технические требования

  • Использовать строковые методы (split(), lower(), replace(), find(), count() и др.)
  • Применять списки и циклы для обработки слов
  • Обрабатывать пустой ввод (выводить сообщение «Текст пуст» и завершать или запрашивать снова)
  • Код должен быть разбит на логические блоки (ввод, обработка, вывод) с комментариями
  • Не использовать готовые функции для статистики текста из сторонних библиотек — всё пишем сами

Критерии качества

  • Понятный интерфейс: подсказки, что вводить
  • Результаты выводятся с пояснениями
  • Программа не завершается с ошибкой при нестандартном вводе (знаки препинания, лишние пробелы, пустые строки)
  • Код аккуратно отформатирован, переменные названы осмысленно

Пример идеи и кода

Ниже показан возможный подход к решению. Он не является обязательным, но поможет начать.

text = input("Введите текст: ")
if text.strip() == "":
    print("Текст пуст, анализ невозможен.")
else:
    # Символы
    chars_total = len(text)
    chars_no_spaces = len(text.replace(" ", ""))
    print("Символов (с пробелами):", chars_total)
    print("Символов (без пробелов):", chars_no_spaces)
    
    # Слова
    words = text.split()
    print("Количество слов:", len(words))
    
    # Предложения (грубая оценка по точкам, воскл. и вопр. знакам)
    sentences = text.replace("!", ".").replace("?", ".").split(".")
    sentences = [s for s in sentences if s.strip() != ""]
    print("Количество предложений:", len(sentences))
    
    # Самое частое слово
    freq = {}
    for w in words:
        w = w.lower().strip(".,!?;:()") # убираем знаки препинания
        if w:
            freq[w] = freq.get(w, 0) + 1
    if freq:
        top_word = max(freq, key=freq.get)
        print("Самое частое слово:", top_word, "(", freq[top_word], "раз)")

Этот пример показывает базовые метрики. Вы можете добавить поиск самого длинного/короткого слова, подсчет уникальных слов и многое другое.

Чеклист перед сдачей

Перед отправкой проекта убедитесь, что:

Функциональность

  • Программа корректно работает с обычным текстом
  • Пустой ввод обрабатывается без ошибок
  • Текст со знаками препинания не ломает подсчёт слов
  • Минимум 4 характеристики выводятся на экран

Код

  • Код написан на Python 3, без синтаксических ошибок
  • Использованы циклы, списки, строковые методы (явно видно применение)
  • Присутствуют комментарии к основным блокам
  • Названия переменных понятны (не a1, b2, а word_count, char_count)

Оформление

  • Файл сохранён как text_analyzer.py
  • В начале файла указаны имя автора и краткое описание
  • Программа работает одинаково при запуске в любой среде (Thonny, IDLE, терминал)

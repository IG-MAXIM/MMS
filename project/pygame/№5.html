Условные операторы в Python

Научитесь управлять ходом программы с помощью if, elif и else

Оператор if

if выполняет блок кода только тогда, когда условие истинно.

Синтаксис

if условие:
    действие

Важно: после условия ставится двоеточие, а вложенный код пишется с отступом (4 пробела).

Пример

age = 18
if age >= 18:
    print("Вы совершеннолетний")

Оператор if-else

if-else задаёт два пути: один, когда условие истинно, и другой — когда ложно.

Синтаксис

if условие:
    действие_если_True
else:
    действие_если_False

Пример: чётность числа

n = int(input("Число: "))
if n % 2 == 0:
    print("Чётное")
else:
    print("Нечётное")

Оператор if-elif-else

Когда нужно проверить несколько условий, используют цепочку if-elif-else. Python проверяет условия по порядку и выполняет первый подошедший блок.

Синтаксис

if условие_1:
    ...
elif условие_2:
    ...
elif условие_3:
    ...
else:
    ... # если ни одно не подошло

Пример: система оценок

score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("Ниже C")

Практические задания

Задание 1: Проверка знака числа

Запросите число и выведите: «положительное», «отрицательное» или «ноль».

num = float(input("Введите число: "))
if num > 0:
    print("Положительное")
elif num < 0:
    print("Отрицательное")
else:
    print("Это ноль!")

Задание 2: Калькулятор скидки

Сумма покупки больше 1000 — скидка 10%, от 500 до 1000 — 5%, иначе скидки нет.

purchase = float(input("Сумма покупки: "))
if purchase > 1000:
    discount = 0.10
elif purchase >= 500:
    discount = 0.05
else:
    discount = 0

final = purchase * (1 - discount)
print(f"Итоговая сумма: {final}")

Задание 3: Вход в систему

Проверьте логин и пароль (например, admin / qwerty).

login = input("Логин: ")
if login == "admin":
    password = input("Пароль: ")
    if password == "qwerty":
        print("Доступ разрешён")
    else:
        print("Неверный пароль")
else:
    print("Пользователь не найден")

Итог урока

Теперь вы знаете:

  • if – выполнение кода по условию
  • if-else – две ветки развития программы
  • if-elif-else – многовариантный выбор
  • Как проверять равенство (==), сравнения (>, <) и комбинировать условия
Главная идея: Условные операторы дают программе «интеллект». Они позволяют принимать решения и реагировать на разные ситуации.

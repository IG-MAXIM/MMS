Введение в Tkinter: создание окна

Научитесь создавать графические приложения с помощью встроенной библиотеки Tkinter и управлять свойствами окна

Первое окно

Tkinter — стандартная библиотека Python для создания графических интерфейсов. Она не требует установки (есть в большинстве сборок Python).

Базовый шаблон

import tkinter as tk
# Создаём главное окно
root = tk.Tk()
# Задаём заголовок
root.title("Моё приложение")
# Запускаем главный цикл
root.mainloop()

Метод title() устанавливает текст в строке заголовка окна. Если не задать, будет пустая строка.

Размеры и положение

Свойство geometry() задаёт размер и положение окна в формате "ШИРИНАxВЫСОТА+ЛЕВО+ВЕРХ".

# Ширина 500, высота 300, отступ слева 100, сверху 50
root.geometry("500x300+100+50")

Можно также запретить изменение размеров окна мышью:

root.resizable(False, False) # ширина и высота заблокированы

Или задать минимальный размер:

root.minsize(300, 200)

Основные свойства окна

geometry

Строка формат: "ШxВ±Л±В"

  • Ш — ширина в пикселях
  • В — высота в пикселях
  • Л — отступ от левого края экрана (может быть отрицательным)
  • В — отступ от верхнего края

Пример: "800x600+200+100" — окно 800×600 в центре экрана (приблизительно).

resizable / minsize

  • resizable(width, height) — разрешает/запрещает растягивать по ширине и высоте.
  • minsize(w, h) — минимальный размер.
  • Можно задать только горизонтальную блокировку: resizable(False, True).

Эти методы помогают контролировать внешний вид окна и защитить макет от случайных изменений.

Практические задания

Задание 1: Создайте окно

Напишите программу, которая создаёт окно с заголовком "Моё первое окно" и размерами 400×300. Запустите её.

import tkinter as tk
root = tk.Tk()
root.title("Моё первое окно")
root.geometry("400x300")
root.mainloop()

Задание 2: Зафиксируйте размер

Измените программу из задания 1 так, чтобы окно нельзя было растянуть мышью.

import tkinter as tk
root = tk.Tk()
root.title("Не растягивается")
root.geometry("400x300")
root.resizable(False, False)
root.mainloop()

Задание 3: Позиционирование

Создайте окно 500×400 с отступом 150 пикселей от левого края и 80 от верхнего.

import tkinter as tk
root = tk.Tk()
root.title("Позиционированное окно")
root.geometry("500x400+150+80")
root.mainloop()

Итог урока

Вы научились:

  • Подключать библиотеку tkinter
  • Создавать главное окно приложения
  • Устанавливать заголовок с помощью title()
  • Задавать размеры и положение через geometry()
  • Блокировать изменение размеров окна (resizable)
Главное: Создание окна — начальный шаг к построению графического интерфейса. Теперь вы можете сделать программу, которая выглядит как настоящее приложение.

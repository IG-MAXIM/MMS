Методы строк в Python

Освойте встроенные инструменты для обработки текста: регистр, поиск, замена и разбиение

Регистр: upper() и lower()

Методы upper() и lower() преобразуют строку в верхний или нижний регистр. Они не изменяют исходную строку, а возвращают новую.

upper()
ВСЕ БУКВЫ БОЛЬШИЕ
lower()
все буквы маленькие
text = "Привет Мир"
print(text.upper()) # ПРИВЕТ МИР
print(text.lower()) # привет мир

Полезно: для сравнения без учёта регистра приводите обе строки к одному регистру.

Поиск и замена: find() и replace()

find() ищет подстроку и возвращает индекс первого вхождения (или -1, если не найдено). replace() заменяет все вхождения одной подстроки на другую.

find()
находит индекс подстроки
replace()
заменяет подстроку
s = "Hello World"
print(s.find("World")) # 6
print(s.find("Python")) # -1 (не найдено)

new_s = s.replace("Hello", "Привет")
print(new_s) # Привет World

Разбиение строки: split()

Метод split() разбивает строку на части по заданному разделителю и возвращает список подстрок. По умолчанию разделитель — пробел, а также несколько пробелов подряд.

Базовое использование

text = "яблоко банан апельсин"
words = text.split()
print(words) # ['яблоко', 'банан', 'апельсин']

Теперь можно обработать каждое слово в цикле.

С другим разделителем

csv = "Иван,Петров,25"
data = csv.split(",")
print(data) # ['Иван', 'Петров', '25']

Удобно для разбора CSV, логов и т.д.

Практические задания

Задание 1: Стандартизация имени

Запросите имя пользователя, которое может быть написано в любом регистре, и выведите его с заглавной буквы (например, "иВАН" → "Иван").

name = input("Введите имя: ")
name = name.lower().capitalize() # или name[0].upper() + name[1:].lower()
print("Привет,", name)

Задание 2: Поиск слова в предложении

Дано предложение. Проверьте, содержит ли оно слово "Python" (без учёта регистра). Выведите его позицию или сообщение об отсутствии.

text = input("Введите текст: ")
pos = text.lower().find("python")
if pos != -1:
    print("Слово 'Python' найдено на позиции", pos)
else:
    print("Слово 'Python' не найдено")

Задание 3: Замена запрещённых слов

Замените в строке все вхождения слова "плохой" на "хороший". Выведите результат.

text = input("Введите текст: ")
new_text = text.replace("плохой", "хороший")
print(new_text)

Задание 4: Инициалы

Пользователь вводит ФИО (три слова через пробел). Выведите фамилию и инициалы (например, "Иванов И. П.").

fio = input("Введите ФИО: ")
parts = fio.split()
if len(parts) == 3:
    lastname, firstname, middlename = parts
    initials = f"{lastname} {firstname[0].upper()}. {middlename[0].upper()}."
    print(initials)
else:
    print("Введите ровно три слова")

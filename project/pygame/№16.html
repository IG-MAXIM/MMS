Методы списков в Python

Изучите встроенные методы для изменения списков: добавление, удаление, сортировка и переворот

Добавление элементов

Методы append() и insert() позволяют расширять список.

append(x)
добавляет x в конец списка
insert(i, x)
вставляет x на позицию i

Примеры

nums = [1, 2, 3]
nums.append(4)
print(nums) # [1, 2, 3, 4]

nums.insert(1, 99)
print(nums) # [1, 99, 2, 3, 4]

Важно: append и insert изменяют список «на месте» и ничего не возвращают (None).

Удаление элементов

Методы remove() и pop() удаляют элементы из списка.

remove(x)
удаляет первое вхождение x
pop(i)
удаляет и возвращает элемент по индексу i (по умолчанию последний)

Примеры

lst = [10, 20, 30, 20]
lst.remove(20) # удаляет первый 20
print(lst) # [10, 30, 20]

n = lst.pop(1) # удаляет и возвращает 30
print(lst) # [10, 20]
print(n) # 30

Осторожно!

remove(x) вызывает ошибку ValueError, если элемент не найден. Проверяйте наличие через in.

Сортировка и переворот: sort() и reverse()

Метод sort() сортирует список по возрастанию (по умолчанию), а reverse() меняет порядок элементов на обратный.

sort()

nums = [5, 2, 8, 1]
nums.sort()
print(nums) # [1, 2, 5, 8]

# По убыванию
nums.sort(reverse=True)
print(nums) # [8, 5, 2, 1]

reverse()

words = ["А", "Б", "В"]
words.reverse()
print(words) # ["В", "Б", "А"]

В отличие от среза [::-1], reverse() тоже изменяет сам список.

Практические задания

Задание 1: Список дел

Создайте пустой список дел (tasks). Добавьте в него три задачи с помощью append(), затем вставьте ещё одну задачу в начало с помощью insert(). Выведите итоговый список.

tasks = []
tasks.append("Погулять")
tasks.append("Сделать уроки")
tasks.append("Приготовить ужин")
tasks.insert(0, "Проснуться")
print(tasks)

Задание 2: Удаление по значению

Дан список [7, 2, 7, 4, 7]. Удалите все вхождения числа 7, используя цикл и remove(). (Подсказка: используйте while 7 in lst:)

lst = [7, 2, 7, 4, 7]
while 7 in lst:
    lst.remove(7)
print(lst) # [2, 4]

Задание 3: Стек через pop

Создайте список из трёх чисел. Используя pop() без аргументов, извлеките все элементы по очереди и выведите их. Обратите внимание на порядок извлечения.

stack = [1, 2, 3]
while len(stack) > 0:
    print(stack.pop())
# Выведет: 3, 2, 1
# (последний пришёл — первый ушёл)

Задание 4: Рейтинг имён

Дан список имён ["Мария", "Алексей", "Елена"]. Отсортируйте его по алфавиту и выведите. Затем переверните список и выведите снова.

names = ["Мария", "Алексей", "Елена"]
names.sort()
print("По алфавиту:", names)
names.reverse()
print("В обратном порядке:", names)

Списки в Python

Создание, доступ к элементам и перебор последовательностей

Создание списков

Список — это упорядоченная коллекция элементов, которые могут быть любого типа. Списки изменяемы: элементы можно добавлять, удалять и менять.

Способы создания

# Пустой список
empty = []

# С элементами
fruits = ["яблоко", "банан", "апельсин"]

# Смешанные типы
mixed = [1, "два", True, 3.14]

Важно: список — это упорядоченная коллекция. Каждый элемент имеет свой номер (индекс).

Доступ к элементам

Для доступа к элементу используется индекс в квадратных скобках. Индексация такая же, как у строк: с 0 и отрицательные с конца.

Прямой индекс
lst[0], lst[1]...
Отрицательный
lst[-1], lst[-2]...
nums = [10, 20, 30, 40]
print(nums[0]) # 10
print(nums[-1]) # 40
print(nums[1:3]) # [20, 30] (срез)

Ошибка!

Если индекс выходит за границы списка, возникает ошибка IndexError. Проверяйте длину через len(lst).

Перебор элементов списка

Списки удобно обрабатывать в цикле for. Есть два основных способа перебора: по элементам и по индексам.

По элементам

fruits = ["яблоко", "банан"]
for item in fruits:
    print(item)

Просто и понятно, когда индекс не важен.

По индексам (range)

for i in range(len(fruits)):
    print(i, fruits[i])

Даёт и индекс, и элемент. Удобно для изменения элементов.

Также можно использовать enumerate() для одновременного получения индекса и значения.

for i, fruit in enumerate(fruits):
    print(f"{i+1}. {fruit}")

Практические задания

Задание 1: Любимые цвета

Создайте список из трёх любимых цветов. Выведите первый и последний цвет списка.

colors = ["синий", "зелёный", "красный"]
print("Первый:", colors[0])
print("Последний:", colors[-1])

Задание 2: Сумма чисел

Дан список чисел [4, 8, 15, 16, 23, 42]. Вычислите и выведите сумму всех элементов.

numbers = [4, 8, 15, 16, 23, 42]
total = 0
for num in numbers:
    total += num
print("Сумма:", total)

Задание 3: Проверка наличия

Запросите у пользователя строку и проверьте, есть ли она в заранее заданном списке покупок.

shop_list = ["хлеб", "молоко", "яйца"]
item = input("Что ищете? ")
if item in shop_list:
    print("Есть в списке")
else:
    print("Нет в списке")

Задание 4: Вывод с нумерацией

Дан список имён. Выведите их в формате "1. Имя", "2. Имя" и т.д. (используйте enumerate()).

names = ["Анна", "Борис", "Вера"]
for i, name in enumerate(names):
    print(f"{i+1}. {name}")

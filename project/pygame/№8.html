Цикл for и функция range() в Python

Освойте повторение действий: итерация по данным, генерация последовательностей и вложенные циклы

Что такое цикл for?

Цикл for выполняет один и тот же блок кода для каждого элемента последовательности (списка, строки, диапазона и т.д.).

for переменная in последовательность:
    действие(переменная)

Для чего нужен?

  • Обработка всех элементов списка
  • Посимвольный перебор строки
  • Повторение действий заданное число раз (через range)
  • Обход данных в словарях и множествах

Функция range()

range() генерирует последовательность целых чисел и часто используется с for для повторения действий определённое количество раз.

range(stop)
0, 1, ..., stop-1
range(start, stop)
от start до stop-1
range(start, stop, step)
с шагом step

Примеры

# 5 повторений: 0,1,2,3,4
for i in range(5):
    print(i)

# От 2 до 5 (не включая 6)
for i in range(2, 6):
    print(i)

# Чётные от 0 до 8
for i in range(0, 10, 2):
    print(i)

Итерация по спискам и строкам

По списку

fruits = ["яблоко", "банан", "апельсин"]
for fruit in fruits:
    print(fruit)

Выведет каждый фрукт на новой строке.

По строке

word = "Python"
for letter in word:
    print(letter)

Выведет каждую букву отдельно.

Задание 1: Сумма чисел от 1 до N

Запросите число N и вычислите сумму всех целых чисел от 1 до N.

N = int(input("Введите N: "))
total = 0
for i in range(1, N + 1):
    total += i
print("Сумма:", total)

number = 5
for i in range(1, 11):

Задание 3: Подсчёт гласных

Дана строка. Посчитайте, сколько в ней гласных букв (а, е, ё, и, о, у, ы, э, ю, я).

text = input("Введите текст: ")
vowels = "аеёиоуыэюя"
count = 0
for ch in text.lower():
    if ch in vowels:
        count += 1
print("Гласных:", count)

Вложенные циклы, break и continue

Вложенные циклы

Цикл внутри другого цикла — для обхода двумерных структур или комбинаций.

for i in range(3):
    for j in range(2):
        print("i =", i, "j =", j)

break и continue

break — немедленно выходит из цикла.
continue — переходит к следующей итерации.

# break: остановка при i == 5
for i in range(10):
    if i == 5:
        break
    print(i)

# continue: пропуск 2
for i in range(5):
    if i == 2:
        continue
    print(i)

Сложные задания

Поиск простых чисел до N

Выведите все простые числа от 2 до заданного N.

N = int(input("Введите N: "))
print("Простые числа от 2 до", N, ":")
for num in range(2, N+1):
    is_prime = True
    for d in range(2, int(num**0.5)+1):
        if num % d == 0:
            is_prime = False
            break
    if is_prime:
        print(num, end=" ")

Прямоугольник из звёздочек

Запросите ширину и высоту и нарисуйте прямоугольник из символов '*'.

w = int(input("Ширина: "))
h = int(input("Высота: "))
for i in range(h):
    for j in range(w):
        print("*", end="")
    print()

Анализ списка чисел

Найдите минимальное, максимальное и среднее арифметическое в списке (задан заранее или вводится).

numbers = [3, 7, 2, 9, 1, 5, 8]
# или ввод: list(map(int, input().split()))
mn = numbers[0]
mx = numbers[0]
total = 0
for x in numbers:
    if x < mn: mn = x
    if x > mx: mx = x
    total += x
avg = total / len(numbers)
print("Мин:", mn, "Макс:", mx, "Сред:", avg)

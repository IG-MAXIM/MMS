Установка Pygame. Инициализация окна, игровой цикл

Начните создавать настоящие 2D-игры: установите библиотеку, создайте окно и оживите его с помощью игрового цикла

Установка и первое окно

Pygame — это набор модулей для создания 2D-игр на Python. Он не входит в стандартную библиотеку, поэтому его нужно установить отдельно.

# В терминале (командной строке):
pip install pygame

После установки можно создавать окно:

import pygame
pygame.init()
screen = pygame.display.set_mode((800, 600))
pygame.display.set_caption("Моя игра")
# ... здесь будет игровой цикл ...
pygame.quit()

set_mode принимает кортеж (ширина, высота). Можно добавить флаги, например pygame.RESIZABLE для изменения размера окна.

Игровой цикл и рисование фигур

Сердце любой Pygame-программы — игровой цикл. Он работает, пока пользователь не закроет окно, и каждый кадр обновляет экран.

running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
    # Рисование
    screen.fill((0, 0, 0)) # чёрный фон
    pygame.draw.rect(screen, (255, 0, 0), (100, 100, 50, 50)) # красный квадрат
    pygame.display.flip() # обновить экран

Цвета задаются в формате RGB: (красный, зелёный, синий), каждое число от 0 до 255.

Другие фигуры:

  • pygame.draw.circle(surface, color, center, radius)
  • pygame.draw.line(surface, color, start_pos, end_pos, width)

Обработка клавиш (стрелки, WASD)

Pygame позволяет реагировать на нажатия клавиш двумя способами: через события KEYDOWN и через состояние клавиатуры pygame.key.get_pressed().

События KEYDOWN

for event in pygame.event.get():
    if event.type == pygame.KEYDOWN:
        if event.key == pygame.K_LEFT:
            x -= 5

Срабатывает один раз при нажатии. Хорошо для дискретных действий (прыжок, выстрел).

Состояние клавиш

keys = pygame.key.get_pressed()
if keys[pygame.K_w]:
    y -= 5 # вверх
if keys[pygame.K_s]:
    y += 5 # вниз

Подходит для плавного движения (персонаж бежит, пока клавиша зажата).

Коды клавиш стрелок: K_LEFT, K_RIGHT, K_UP, K_DOWN. Для WASD — K_w, K_a, K_s, K_d.

Практические задания

Задание 1: Создайте окно

Установите Pygame и напишите программу, которая создаёт окно 600×400 с заголовком «Моё первое окно» и синим фоном. Окно должно висеть до тех пор, пока его не закроют.

import pygame
pygame.init()
screen = pygame.display.set_mode((600, 400))
pygame.display.set_caption("Моё первое окно")
running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
    screen.fill((0, 0, 255))
    pygame.display.flip()
pygame.quit()

Задание 2: Нарисуйте фигуры

В окне 500×400 нарисуйте зелёный прямоугольник (x=50, y=30, ширина 80, высота 50), красный круг (центр 200, 200, радиус 40) и белую линию от (300, 100) до (400, 300) толщиной 4.

import pygame
pygame.init()
screen = pygame.display.set_mode((500, 400))
running = True
while running:
    for e in pygame.event.get():
        if e.type == pygame.QUIT: running = False
    screen.fill((0,0,0))
    pygame.draw.rect(screen, (0,255,0), (50,30,80,50))
    pygame.draw.circle(screen, (255,0,0), (200,200), 40)
    pygame.draw.line(screen, (255,255,255), (300,100), (400,300), 4)
    pygame.display.flip()
pygame.quit()

Задание 3: Управление квадратом

Создайте окно 600×400 с белым квадратом (размер 40×40) в центре. Реализуйте движение квадрата стрелками клавиатуры (или WASD) на 5 пикселей за кадр, используя get_pressed(). Не дайте квадрату выйти за границы окна.

import pygame
pygame.init()
w, h = 600, 400
screen = pygame.display.set_mode((w, h))
x, y = w//2, h//2
speed = 5
clock = pygame.time.Clock()
running = True
while running:
    for e in pygame.event.get():
        if e.type == pygame.QUIT: running = False
    keys = pygame.key.get_pressed()
    if keys[pygame.K_LEFT] or keys[pygame.K_a]: x -= speed
    if keys[pygame.K_RIGHT] or keys[pygame.K_d]: x += speed
    if keys[pygame.K_UP] or keys[pygame.K_w]: y -= speed
    if keys[pygame.K_DOWN] or keys[pygame.K_s]: y += speed
    # Границы
    x = max(20, min(w-20, x))
    y = max(20, min(h-20, y))
    screen.fill((0,0,0))
    pygame.draw.rect(screen, (255,255,255), (x-20, y-20, 40, 40))
    pygame.display.flip()
    clock.tick(60)
pygame.quit()

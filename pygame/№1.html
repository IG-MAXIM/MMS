История создания

Python создан в 1989–1991 годах Гвидо ван Россумом. Вдохновлён проектом ABC, учёл важность обратной связи и сообщества.

1989 — Начало разработки Python
20 февраля 1991 — Первая публичная версия
2000 — Python 2.0 с поддержкой Unicode
2008 — Python 3.0 (несовместим с Python 2)
2020 — Окончание поддержки Python 2.7

Особенности Python

Высокоуровневый, интерпретируемый, с читаемым синтаксисом.

Первый шаг — установка Python

Инструкция для Windows

# Для Windows:
1. Скачайте Python с python.org
2. Запустите установщик
3. Отметьте "Add Python to PATH"
4. Нажмите "Install Now"

# Проверка установки:
Откройте командную строку и введите:
python --version
# Должна появиться версия Python

Функции ввода и вывода в Python

Научитесь общаться с программой: выводить данные и получать ответ от пользователя

Функция print()

Функция print() выводит информацию на экран. Её можно настроить так, как удобно вам.

Параметры

значения
данные через запятую
sep
разделитель (по умолчанию пробел)
end
что выводить в конце (по умолчанию перевод строки)

Разделитель sep

print("01", "02", "2024", sep=".")

Выведет: 01.02.2024

Окончание end

print("Привет", end=" ")
print("мир!")

Выведет: Привет мир!

Функция input()

Функция input() получает от пользователя текстовую строку. Чтобы получить число, нужно преобразовывать.

Важно!

input() всегда возвращает строку. Для чисел используйте int() или float().

Преобразование типов

int()

возвращает целое число

float()

возвращает дробное число

str()

возвращает строку (обычно не нужна, input и так строка)

Частая ошибка

age = input("Возраст: ")
# ОШИБКА: age – строка, + 5 нельзя

age = int(input("Возраст: "))
# ПРАВИЛЬНО: age теперь число

Практические задания

Задание 1: Вывод информации о себе

Выведите три строчки: имя, класс и любимый предмет.

print("Меня зовут: Анна")
print("Я учусь в 8 классе")
print("Мой любимый предмет: информатика")

Задание 2: Использование переменных

Создайте переменные и выведите их одной командой.

name = "Алексей"
grade = 8
subject = "математика"
hobby = "программирование"

print("Имя:", name, "Класс:", grade, "Предмет:", subject, "Хобби:", hobby)

Задание 3: Сумма двух чисел (ввод с клавиатуры)

Запросите два числа и выведите их сумму.

num1 = int(input("Введите первое число: "))
num2 = int(input("Введите второе число: "))
print("Сумма:", num1 + num2)

Задание 4: Периметр прямоугольника

Запросите длину и ширину, вычислите периметр.

length = float(input("Длина: "))
width = float(input("Ширина: "))
print("Периметр:", 2 * (length + width))

Задание 5: Визитная карточка

Создайте программу-генератор визитной карточки.

print("=== Генератор визитной карточки ===")
name = input("ФИО: ")
school = input("Школа: ")
phone = input("Телефон: ")

print("="*30)
print("ВИЗИТНАЯ КАРТОЧКА")
print("="*30)
print("ФИО:", name)
print("Школа:", school)
print("Телефон:", phone)
print("="*30)

Цикл while в Python

Изучите циклы с условием, бесконечные циклы и управление повторениями

Что такое цикл while?

Цикл while выполняет блок кода до тех пор, пока условие остаётся истинным.

while условие:
    действие

Проверка условия происходит перед каждой итерацией. Как только условие становится ложным, цикл завершается.

Простой пример

i = 1
while i <= 5:
    print(i)
    i += 1

Выведет числа от 1 до 5.

Бесконечный цикл и break

Если условие всегда истинно, цикл становится бесконечным. Для выхода используется оператор break.

while True:
бесконечный цикл
break
немедленный выход

Пример: вечный ввод

while True:
    cmd = input("Введите команду (exit для выхода): ")
    if cmd == "exit":
        break
    print("Вы ввели:", cmd)

Пропуск итерации и вложенные условия

continue

Пропускает оставшуюся часть тела цикла и переходит к следующей проверке условия.

i = 0
while i < 5:
    i += 1
    if i == 3:
        continue
    print(i)

Выведет: 1 2 4 5 (3 пропущено).

Сложное условие

Можно комбинировать while с логическими операторами для гибкого управления.

# Пока x в диапазоне (0, 100)
x = 50
while 0 < x < 100:
    x = int(input("Новое значение: "))
print("Вышли за границы")

Практические задания

Задание 1: Угадай число

Программа загадывает число (например, 7), а пользователь пытается угадать. Используйте while и break.

secret = 7
while True:
    guess = int(input("Угадайте число: "))
    if guess == secret:
        print("Верно!")
        break
    else:
        print("Не угадали, попробуйте снова.")

Задание 2: Сумма цифр (до нуля)

Пользователь вводит числа. Программа суммирует их до тех пор, пока не будет введён 0. Используйте while.

total = 0
num = int(input("Введите число (0 – стоп): "))
while num != 0:
    total += num
    num = int(input("Введите число (0 – стоп): "))
print("Сумма:", total)

Задание 3: Проверка пароля

Запрашивайте пароль, пока пользователь не введёт "qwerty". Используйте бесконечный цикл и break.

while True:
    psw = input("Введите пароль: ")
    if psw == "qwerty":
        print("Доступ разрешён")
        break
    else:
        print("Неверный пароль")

Итог урока

Теперь вы знаете:

  • Цикл while — повторение, пока условие истинно
  • Бесконечные циклы с while True
  • Выход из цикла с помощью break
  • Пропуск итерации через continue
Главное: Цикл while незаменим, когда количество повторений заранее неизвестно и зависит от условия.

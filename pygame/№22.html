Виджеты Label, Button и обработка события command

Научитесь добавлять текст и кнопки в приложение, а также обрабатывать нажатия с помощью функций

Виджет Label

Label — это простая текстовая метка (надпись), которая отображает строку на форме. Её можно настроить: шрифт, цвет, выравнивание и многое другое.

Создание и размещение

import tkinter as tk
root = tk.Tk()
# Создаём метку с текстом
label = tk.Label(root, text="Привет, Tkinter!")
# Размещаем её в окне (упаковщик pack)
label.pack()
root.mainloop()

Основные параметры:

  • text — отображаемый текст
  • font — шрифт (например, ("Arial", 14))
  • fg / foreground — цвет текста
  • bg / background — цвет фона

Виджет Button и событие command

Button — это кликабельная кнопка. При нажатии она вызывает функцию, указанную в параметре command.

Простая кнопка

def say_hello():
    print("Привет!")

btn = tk.Button(root, text="Нажми", command=say_hello)
btn.pack()

Функция, переданная в command, вызывается без аргументов. Если нужно передать аргументы, используют lambda:

btn = tk.Button(root, text="Аргументы",
    command=lambda: say_hello_to("Мир"))

Взаимодействие Label и Button

Одна из самых частых задач — изменение текста метки при нажатии кнопки. Для этого нужно обновлять свойство label["text"] или использовать метод label.config(text=...).

Изменение свойства text

def update_label():
    label["text"] = "Новый текст"
# или label.config(text="Новый текст")

Счётчик нажатий

counter = 0
def count_click():
    global counter
    counter += 1
    label.config(text=f"Нажатий: {counter}")

btn = tk.Button(root, text="Жми!", command=count_click)

Практические задания

Задание 1: Приветствие по кнопке

Создайте окно с меткой "Нажми кнопку" и кнопкой "Поздороваться". При нажатии на кнопку текст метки должен измениться на "Привет!".

import tkinter as tk
root = tk.Tk()
root.title("Приветствие")
def greet():
    label.config(text="Привет!")
label = tk.Label(root, text="Нажми кнопку")
label.pack()
btn = tk.Button(root, text="Поздороваться", command=greet)
btn.pack()
root.mainloop()

Задание 2: Счётчик кликов

Создайте метку "Нажатий: 0" и кнопку "Добавить". При каждом нажатии счётчик должен увеличиваться на 1.

import tkinter as tk
root = tk.Tk()
root.title("Счётчик")
count = 0
def increment():
    global count
    count += 1
    label.config(text=f"Нажатий: {count}")
label = tk.Label(root, text="Нажатий: 0")
label.pack(pady=10)
btn = tk.Button(root, text="Добавить", command=increment)
btn.pack()
root.mainloop()

Задание 3: Смена цвета фона

Создайте окно с меткой и двумя кнопками: "Красный" и "Синий". При нажатии на кнопку должен меняться фоновый цвет метки (или всего окна).

import tkinter as tk
root = tk.Tk()
root.title("Цветной фон")
def set_red():
    root.config(bg="red")
def set_blue():
    root.config(bg="blue")
label = tk.Label(root, text="Меняй цвет окна")
label.pack(pady=10)
btn_red = tk.Button(root, text="Красный", command=set_red)
btn_red.pack()
btn_blue = tk.Button(root, text="Синий", command=set_blue)
btn_blue.pack()
root.mainloop()

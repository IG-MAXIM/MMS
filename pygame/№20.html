Что такое модуль?

Модуль — это файл с кодом Python (расширение .py), который можно подключить к своей программе и использовать написанные в нём функции, классы и переменные.

Библиотека — это набор модулей, объединённых общей темой (например, математические операции, работа с графикой).

Модуль = Ящик с инструментами Открыл — и пользуешься готовыми функциями

Python поставляется с большой стандартной библиотекой: math, random, datetime и многими другими. Их не нужно устанавливать отдельно.

Импорт модуля

Чтобы использовать модуль, его нужно импортировать с помощью ключевого слова import.

import math # Импорт всего модуля
print(math.sqrt(25)) # 5.0

Можно импортировать только нужные функции:

from math import sqrt, pi
print(sqrt(16)) # 4.0
print(pi) # 3.1415926535...

Или дать модулю короткое имя (псевдоним):

import math as m
print(m.cos(0)) # 1.0

Модуль math: главные функции и константы

math предоставляет математические функции и константы. Вот самые популярные из них:

Функции

  • sqrt(x) — квадратный корень
  • ceil(x) — округление вверх
  • floor(x) — округление вниз
  • sin(x), cos(x) — синус/косинус (угол в радианах)
  • radians(x) — перевод градусов в радианы
  • pow(x, y) — x в степени y
  • fabs(x) — модуль (абсолютное значение) числа

Константы

  • pi — число π (3.141592...)
  • e — число Эйлера (2.718281...)
  • tau — число τ = 2π (6.283185...)
  • inf — бесконечность

Константы используются как обычные переменные: 2 * math.pi * r.

Практические задания

Задание 1: Квадратный корень

Запросите у пользователя число и выведите его квадратный корень с помощью math.sqrt(). Если число отрицательное, выведите сообщение об ошибке.

import math
x = float(input("Введите число: "))
if x >= 0:
    print("Корень:", math.sqrt(x))
else:
    print("Ошибка: нельзя извлечь корень из отрицательного числа")

Задание 2: Площадь круга

Запросите радиус круга и вычислите его площадь по формуле π * r². Используйте math.pi и math.pow().

import math
r = float(input("Радиус: "))
area = math.pi * math.pow(r, 2)
print("Площадь круга:", round(area, 2))

Задание 3: Округление

Дано число 4.3. Выведите его округление вверх (ceil) и вниз (floor). Объясните разницу.

from math import ceil, floor
num = 4.3
print("ceil:", ceil(num)) # 5
print("floor:", floor(num)) # 4

Задание 4: Синус угла

Запросите угол в градусах, переведите в радианы с помощью math.radians() и вычислите его синус.

import math
deg = float(input("Угол в градусах: "))
rad = math.radians(deg)
print("Синус:", round(math.sin(rad), 4))

Строки: индексация, срезы, длина строки

Узнайте, как обращаться к отдельным символам, извлекать части строк и измерять их длину

Индексация строк

Каждый символ строки имеет свой индекс – порядковый номер, начинающийся с 0. Индексы можно использовать для получения одного символа.

Прямой индекс
s[0], s[1], s[2]…
Отрицательный индекс
s[-1] – последний, s[-2] – предпоследний

Пример

word = "Python"
print(word[0]) # P
print(word[-1]) # n
print(word[2]) # t

Длина строки len()

Функция len() возвращает количество символов в строке.

word = "Python"
print(len(word)) # 6

Длина нужна, чтобы:

  • Определить последний индекс
  • Проверить, не пустая ли строка
  • Ограничить перебор в цикле

Важно: последний индекс строки = len(s) - 1.

Срезы строк

Срез позволяет извлечь часть строки. Синтаксис: строка[начало:конец:шаг].

Параметры среза

  • start – начальный индекс (включительно)
  • stop – конечный индекс (не включается)
  • step – шаг (по умолчанию 1)

Если что-то не указано, берутся крайние значения (0, длина, 1).

Примеры срезов

s = "Hello World"
s[0:5] # Hello
s[6:] # World
s[:5] # Hello
s[::2] # HloWrd
s[::-1] # dlroW olleH (переворот)

Практические задания

Задание 1: Первый и последний символ

Запросите строку и выведите её первый и последний символ.

s = input("Введите строку: ")
print("Первый символ:", s[0])
print("Последний символ:", s[-1])

Задание 2: Извлечение середины

Дана строка. Если её длина чётная, выведите два центральных символа; если нечётная — один центральный.

s = input("Введите строку: ")
n = len(s)
if n % 2 == 0:
    mid = n // 2
    print("Два центральных символа:", s[mid-1:mid+1])
else:
    mid = n // 2
    print("Центральный символ:", s[mid])

Задание 3: Проверка на палиндром

Проверьте, читается ли строка одинаково слева направо и справа налево. Используйте срез.

s = input("Введите строку: ")
if s == s[::-1]:
    print("Это палиндром!")
else:
    print("Не палиндром")

Итог урока

Теперь вы знаете:

  • Как обращаться к символам через индексы (в т.ч. отрицательные)
  • Как получать длину строки функцией len()
  • Как извлекать части строки с помощью срезов [start:stop:step]
  • Как перевернуть строку срезом [::-1]
Главная идея: Умение работать со строками на уровне символов и подстрок – основа для анализа текста, парсинга данных и многих алгоритмов.

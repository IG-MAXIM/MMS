Логические операторы в Python

Научитесь строить сложные условия с помощью and, or и not

Оператор and

Оператор and возвращает True только тогда, когда оба условия истинны.

Таблица истинности

ABA and B
TrueTrueTrue
TrueFalseFalse
FalseTrueFalse
FalseFalseFalse

Примеры

age = 20
has_ticket = True
if age >= 18 and has_ticket:
    print("Доступ разрешён")

Проверка попадания числа в диапазон:

x = 5
if x > 0 and x < 10:
    print("Число в диапазоне (0, 10)")

Оператор or

Оператор or возвращает True, если хотя бы одно условие истинно.

Таблица истинности

ABA or B
TrueTrueTrue
TrueFalseTrue
FalseTrueTrue
FalseFalseFalse

Примеры

day = "суббота"
if day == "суббота" or day == "воскресенье":
    print("Выходной!")

Проверка на несколько допустимых ответов:

answer = input("Хотите продолжить? (да/yes): ")
if answer == "да" or answer == "yes":
    print("Продолжаем!")

Оператор not и приоритет операций

Оператор not

Инвертирует булево значение: not True даёт False и наоборот.

is_banned = False
if not is_banned:
    print("Добро пожаловать!")

Удобно, когда нужно проверить отсутствие свойства.

Приоритет

Порядок вычисления логических операторов:

  1. not – самый высокий
  2. and
  3. or

Используйте скобки (), чтобы управлять порядком.

# Без скобок: not применяется только к x > 0
if not x > 0 and y > 0:
    ...

# Со скобками: явно задаём логику
if not (x > 0 and y > 0):
    ...

Практические задания

Задание 1: Проверка диапазона

Запросите число и проверьте, находится ли оно в диапазоне от 10 до 20 включительно.

num = int(input("Введите число: "))
if num >= 10 and num <= 20:
    print("Число в диапазоне [10, 20]")
else:
    print("Число вне диапазона")

Задание 2: Високосный год

Год является високосным, если он кратен 4, но не кратен 100, либо кратен 400. Реализуйте проверку.

year = int(input("Введите год: "))
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print("Високосный год")
else:
    print("Не високосный")

Задание 3: Проверка строки

Запросите строку и определите, пустая ли она. Используйте not.

text = input("Введите текст: ")
if not text:
    print("Вы ничего не ввели")
else:
    print("Спасибо! Ваш текст:", text)

Задание 4: Комбинированная проверка

Проверьте, что пользователь ввёл "да" или "yes" (регистр не важен) и при этом он не забанен (переменная banned = False).

answer = input("Хотите продолжить? ").lower()
banned = False
if (answer == "да" or answer == "yes") and not banned:
    print("Продолжаем работу!")
else:
    print("Отмена или доступ запрещён")

Итог урока

Теперь вы знаете:

  • and – оба условия истинны
  • or – хотя бы одно истинно
  • not – инверсия значения
  • Приоритет операторов и использование скобок
Главная идея: Логические операторы позволяют делать условия гибкими и точными. Это фундамент для построения сложной логики в ваших программах.

Оператор continue и вложенные циклы

Управление повторениями: пропуск итераций и многомерные обходы данных

Оператор continue

continue немедленно переходит к следующей итерации цикла, пропуская оставшиеся инструкции в текущем шаге.

Пример в for

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Выведет только нечётные числа: 1 3 5 7 9.

Пример в while

i = 0
while i < 10:
    i += 1
    if i == 5:
        continue
    print(i)

Выведет 1 2 3 4 6 7 8 9 10 (5 пропущено).

Вложенные циклы

Один цикл внутри другого — для перебора всех комбинаций или двумерных структур.

Синтаксис

for i in range(3):
    for j in range(3):
        print(i, j)

Таблица умножения

for i in range(1, 6):
    for j in range(1, 6):
        print(i*j, end=" ")
    print()

Выведет таблицу умножения 5×5.

continue во вложенных циклах и break

continue во вложенном цикле

continue действует только на ближайший цикл, в котором он написан.

for i in range(3):
    for j in range(5):
        if j % 2 == 0:
            continue
        print(i, j)

Выводит пары, где j – нечётное.

break во вложенном цикле

Аналогично break прерывает только текущий внутренний цикл.

for i in range(3):
    for j in range(5):
        if j == 2:
            break
        print(i, j)

Выводит для каждого i значения j=0 и j=1.

Практические задания

Задание 1: Пропуск отрицательных

Дан список чисел. Выведите только положительные, используя continue.

nums = [3, -1, 0, 7, -5, 4]
for x in nums:
    if x <= 0:
        continue
    print(x)

Задание 2: Все пары чисел

Выведите все возможные пары (a, b), где a от 1 до 3, b от 1 до 3. Используйте вложенные циклы.

for a in range(1, 4):
    for b in range(1, 4):
        print("(", a, ",", b, ")", end=" ")
    print()

Задание 3: Прямоугольник с пропуском

Нарисуйте прямоугольник 5×5 из звёздочек, но для i=2 пропустите всю строку с помощью continue.

for i in range(5):
    if i == 2:
        continue
    for j in range(5):
        print("*", end="")
    print()

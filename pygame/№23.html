Виджет Entry и компоновка pack, grid

Освойте однострочный ввод текста и управляйте расположением элементов в окне

Виджет Entry

Entry — поле для ввода одной строки текста. Его используют для получения данных от пользователя: имени, возраста, URL и т.д.

Создание и получение текста

entry = tk.Entry(root)
entry.pack()
# Получить введённый текст:
text = entry.get()
# Очистить поле:
entry.delete(0, tk.END)

Параметры:

  • width — ширина в символах
  • show — маскировка (например, "*" для пароля)
  • state — "normal", "readonly", "disabled"

Компоновка pack

pack() — простейший менеджер геометрии. Он размещает виджеты по очереди: друг за другом по вертикали или горизонтали.

Основные параметры

widget.pack(side=tk.TOP, fill=tk.X, padx=5, pady=5)
  • side — TOP, BOTTOM, LEFT, RIGHT
  • fill — X (по ширине), Y (по высоте), BOTH
  • expand — True/False, растягивать ли при изменении окна
  • padx / pady — внешние отступы

Удобен для быстрой вёрстки, но сложные макеты делать трудно.

Компоновка grid

grid() размещает виджеты в ячейках таблицы (строках и столбцах). Это самый гибкий и часто используемый способ.

Базовый синтаксис

label = tk.Label(root, text="Имя:")
label.grid(row=0, column=0)
entry = tk.Entry(root)
entry.grid(row=0, column=1)

Объединение ячеек

# Кнопка на два столбца
btn.grid(row=1, column=0, columnspan=2, sticky="ew")

Параметры:

  • row, column — строка и столбец
  • columnspan, rowspan — на сколько столбцов/строк растянуть
  • sticky — N, S, E, W или их комбинации ("ew", "nsew")
  • padx / pady — отступы внутри ячейки

Практические задания

Задание 1: Копирование текста

Создайте поле Entry и кнопку "Показать". При нажатии на кнопку введённый текст должен появиться в Label под полем.

import tkinter as tk
root = tk.Tk()
def show_text():
    text = entry.get()
    label.config(text=text)
entry = tk.Entry(root)
entry.pack()
btn = tk.Button(root, text="Показать", command=show_text)
btn.pack()
label = tk.Label(root, text="")
label.pack()
root.mainloop()

Задание 2: Форма входа (grid)

С помощью grid() сверстайте форму входа: "Логин" (Label + Entry), "Пароль" (Label + Entry с маскировкой), кнопка "Войти". При нажатии на кнопку сверяйте пароль с захардкоженным значением и выводите результат в отдельную метку.

import tkinter as tk
root = tk.Tk()
root.title("Форма входа")

tk.Label(root, text="Логин:").grid(row=0, column=0, sticky="e", padx=5, pady=5)
entry_login = tk.Entry(root)
entry_login.grid(row=0, column=1, padx=5, pady=5)

tk.Label(root, text="Пароль:").grid(row=1, column=0, sticky="e", padx=5, pady=5)
entry_pass = tk.Entry(root, show="*")
entry_pass.grid(row=1, column=1, padx=5, pady=5)

result_label = tk.Label(root, text="")
result_label.grid(row=2, column=0, columnspan=2, pady=5)

def login():
    if entry_login.get() == "admin" and entry_pass.get() == "123":
        result_label.config(text="Успешно!")
    else:
        result_label.config(text="Неверно!")

tk.Button(root, text="Войти", command=login).grid(row=3, column=0, columnspan=2, pady=10)
root.mainloop()

Задание 3: Калькулятор (grid)

Создайте простой калькулятор: два поля Entry (числа), метка с результатом и 4 кнопки (+, -, *, /), расположенные в таблице 2×2 с помощью grid.

import tkinter as tk
root = tk.Tk()
root.title("Калькулятор")

entry1 = tk.Entry(root, width=10)
entry1.grid(row=0, column=0, columnspan=2, padx=5, pady=5)
entry2 = tk.Entry(root, width=10)
entry2.grid(row=1, column=0, columnspan=2, padx=5, pady=5)

result_label = tk.Label(root, text="Результат")
result_label.grid(row=2, column=0, columnspan=2, pady=5)

def calc(op):
    try:
        a = float(entry1.get())
        b = float(entry2.get())
        if op == "+": res = a + b
        elif op == "-": res = a - b
        elif op == "*": res = a * b
        elif op == "/": res = a / b if b != 0 else "Ошибка"
        result_label.config(text=str(res))
    except:
        result_label.config(text="Введите числа")

tk.Button(root, text="+", command=lambda: calc("+")).grid(row=3, column=0, sticky="ew", padx=2)
tk.Button(root, text="-", command=lambda: calc("-")).grid(row=3, column=1, sticky="ew", padx=2)
tk.Button(root, text="*", command=lambda: calc("*")).grid(row=4, column=0, sticky="ew", padx=2)
tk.Button(root, text="/", command=lambda: calc("/")).grid(row=4, column=1, sticky="ew", padx=2)
root.mainloop()
